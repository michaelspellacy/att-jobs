Browser Support

Please visit http://www.tmp.com/web-standards/ for a listing of the latest browsers that are tested against.

Please note that IE8 falls under limited support.

HTML Specification

HTML5 using XML syntax. All elements must be closed and all attributes quoted.

CSS

All CSS was written in SASS (Syntactically Awesome Stylesheets), a CSS pre-processor. Learn more about SASS at http://sass-lang.com/

JavaScript Libraries and Dependencies

  • jQuery 1.x - JavaScript library that simplifies HTML document traversing, event handling, animating, and Ajax interactions for rapid web development. This system pulls in latest version of jQuery 1.x directly from Google's CDN
  • Modernizer (Custom) - JavaScript library that detects HTML5 and CSS3 features in the user's browser.
  • Respond - Fast & lightweight polyfill for min/max-width CSS3 Media Queries for IE 6-8, and more (if you must have support here).
  • ie7-js - A JavaScript library to make MSIE behave like a standards-compliant browser. (Sigh)

jQuery Plugins

  • Flexslider 2 - The best responsive slider. Period. This is a jQuery plugin.

AddThis

This system pulls in social network enhancement, AddThis, on as as needed basis. See build/js/global.js for more details.

Screen Resolution

Site optimized for a screen resolution of 952 pixels and below. Design will center in viewports higher than this number.

Breakpoints

There are currently 8 breakpoints used across this system:

  • 1024: uincron
  • 940: vectorsigma
  • 840: ultramagnus
  • 768: optimus
  • 600: megatron
  • 540: soundwave
  • 480: cliffjumper
  • 320: frenzy

It is easier to remember breakpoints when they are defined by names (I happen to like Transformers).

Yes. The account number is UA-4300144-6.

Typography

Important: The site uses rem ("root em") units divided by a single context of 16px (100%). Since compounding is no longer an issue, as it is with ems, all font sizes can be divided by this single context.

.page has been reset to have a base font size of 12px (12px / 16px = 0.75rem). All fonts will now render at this size unless specified differently. Despite this new font size, all children elements are divided by the root size of 16px.

Font Stack

Base font family is Arial, "Helvetica Neue", Helvetica, sans-serif

Images

The following image types need to be provided in the following sizes:

Hero Module Graphic

The primary image used on Module: Single.

  • Type: JPG
  • Size: 1506 x 1506
  • Quality: 0
  • Progressive: Yes
  • Naming Convention: module-single-[unique-name].jpg

Vertical Module Graphic

Vertical image used on Module: Triple Media Right.

  • Type: JPG
  • Size: 228 x 468
  • Quality: ~72
  • Progressive: Yes
  • Naming Convention: vz-[unique-name or incremental number].jpg

Horizontal Module Graphic

Horizontal image used on Module: Triple Media Top.

  • Type: JPG
  • Size: 468 x 228
  • Quality: ~72
  • Progressive: Yes
  • Naming Convention: hz-[unique-name or incremental number].jpg

Single Module or Video Graphic

  • Type: JPG or GIF
  • Size: 228 x 228
  • Quality: ~72 (if JPG)
  • Progressive: Yes
  • Naming Convention: single-[unique-name or incremental number].jpg or single-video-[unique-name or incremental number].jpg

Awards (Slider)

  • Type: GIF or JPG
  • Size: 168 x 102
  • Transparent: No
  • Naming Convention: awards-[unique-name]-sm.gif

Tip: Unique names can benefit acessibility and SEO.

Note: Due to nature of award image source, gradient background is used.

Awards (Single)

  • Type: GIF or JPG
  • Size: 172 x 146
  • Transparent: No
  • Naming Convention: awards-[unique-name]-lg.gif
